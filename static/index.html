Eskalate Movie API

REST API for a personal movie collection

Available Endpoints

Authentication

POST /api/auth/login

Login with email and password

POST /api/auth/signup

Register a new user with email, username, and password

Movies

GET /api/movies

Get a paginated list of movies

POST /api/movies

Create a new movie (auth required)

GET /api/movies/search

Search movies by title (case-insensitive)

GET /api/movies/{id}

Get details for a single movie by ID

PUT /api/movies/{id}

Update a movie (auth required, must own movie)

DELETE /api/movies/{id}

Delete a movie (auth required, must own movie)
